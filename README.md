# Functional_Testing
# MDPI 2022 : Functional Testing of DNN Accelerators

## Envrionment

* Python 3.7.13
* Numpy 1.21.5
* Pandas 1.3.5
* Matplotlib 3.2.2
* Torch 1.10.0+cu111
